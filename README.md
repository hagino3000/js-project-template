# Web-Frontend template with asset pipeline

Using

- eslint
  - JavaScript code linting
- webpack
  - Generate assets
  - Replace `src` attribute of script tag with build revision
- nodemon
  - Auto build when source file edited
- simple-autoreload-server
  - Auto reload
- html5 boilerplate
  - HTML template

## Setup

Requirements

- Nodejs

```
make setup
```

## Build

```
make build
```
